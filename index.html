Página única

Esta es una página única para móvil creada a partir de una plantilla boilerplate que puedes copiar para construir tu primer sitio con JQuery Mobile. Cada vínculo o formulario "empujará" una nueva página por medio de Ajax para dar soporte a las transiciones de página animadas.

Para empezar, sólo ve el código fuente, cópialo y modifícalo. Todos los CSS y JS están vinculados a las versiones más nuevas de JQuery, así que es facilísimo de usar. Recuerda agregar una etiqueta meta para incluir el atributo viewport y ajustar el nivel de acercamiento (zoom) inicial.

Esta plantilla es un documento HTML estándar, con un contenedor page único, a diferencia de la plantilla multi-página, que tiene múltiples páginas dentro de ella. Te recomiendo que construyas tu sitio o aplicación como una serie de páginas únicas como ésta, ya que será un documento más limpio, ligero y trabajarás mejor sin JavaScript.
